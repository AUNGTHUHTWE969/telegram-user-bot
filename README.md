# Telegram User Info Bot

မြန်မာလို ရေးထားတဲ့ Telegram User Info Bot - Render.com မှာ 24/7 အလုပ်လုပ်မယ်

## Features
- User information lookup
- ID checking
- Group management tools
- 24/7 uptime on Render

## Setup on Render
1. Fork this repository
2. Create account on [Render.com](https://render.com)
3. Connect your GitHub repository
4. Add `BOT_TOKEN` environment variable
5. Deploy!

## Bot Commands
- `/start` - Start the bot
- `/info` - Get user info
- `/myid` - Get your user ID
- `/chatid` - Get chat ID
- `/groupinfo` - Get group info
- `/status` - Bot status
- `/help` - Help message
